# マンチェスターのコンピュータ専攻学生数

**テーブル構造:** `nss(ukprn, institution, subject, level, question, response, sample, score, A_STRONGLY_DISAGREE, A_DISAGREE, A_NEUTRAL, A_AGREE, A_STRONGLY_AGREE)`

マンチェスターの大学のQ01における総サンプル数とコンピュータ専攻((8)%)の学生数を表示
```sql
SELECT institution, SUM(sample), SUM(CASE WHEN subject LIKE '(8)%' THEN sample END) comp FROM nss 
WHERE question='Q01' AND (institution LIKE '%Manchester%') 
GROUP BY institution
```
