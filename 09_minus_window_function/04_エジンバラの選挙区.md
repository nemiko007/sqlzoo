# エジンバラの選挙区

**テーブル構造:** `ge(id, lastName, firstName, constituency, party, votes, yr)`

2017年のエジンバラ周辺（S14000021〜S14000026）の選挙区ごとの順位を表示
```sql
SELECT constituency, party, votes, RANK() OVER (PARTITION BY constituency ORDER BY votes DESC) AS posn FROM ge 
WHERE constituency BETWEEN 'S14000021' AND 'S14000026' 
  AND yr = 2017 
ORDER BY posn, constituency
```
