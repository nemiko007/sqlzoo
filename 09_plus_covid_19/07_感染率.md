# 感染率

**テーブル構造:** `covid(name, whn, confirmed, deaths, recovered)`

人口1000万以上の国を対象に、感染率（人口10万あたり）の順位を表示
```sql
SELECT world.name, ROUND(100000*confirmed/population, 2), RANK() OVER (ORDER BY 100000*confirmed/population) AS rank FROM covid 
JOIN world ON covid.name=world.name 
WHERE whn = '2020-04-20' AND population > 10000000 
ORDER BY population DESC
```
