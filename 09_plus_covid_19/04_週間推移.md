# 週間推移

**テーブル構造:** `covid(name, whn, confirmed, deaths, recovered)`

イタリアの2020年の月曜日のみの新規感染者数を表示
```sql
SELECT name, DATE_FORMAT(whn,'%Y-%m-%d'), confirmed - LAG(confirmed, 1) OVER (ORDER BY whn) AS "new this week" FROM covid 
WHERE name='Italy' AND WEEKDAY(whn) = 0 AND YEAR(whn) = 2020 
ORDER BY whn
```
